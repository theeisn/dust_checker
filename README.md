# dust_checker
## AutoHotkey script to check the dust value of uniques in Path of Exile 1.

I wanted an easy to use tool that can also be customized to make the tooltip bigger so I don't have to squint.

**Requirement**
* AutoHotkey v1.1 installed

**How to install**
1. Download the latest Source Code archive from the [releases](https://github.com/theeisn/dust_checker/releases) page
2. Extract everything in the same folder
3. Double click on the_eisn's dust checker.ahk

**How to use**
* Press CTRL+SHIFT+D while hovering over an identified unique item. A popup will display with the value for a short time.
* The dust value used is the one available on poedb.tw, so it is not programatically calculated based on the real ilvl of the item or the values of its modifiers;
* Furthermore, Foulborn uniques will have different expected values because they have different mods, but, in general, it should be close.
